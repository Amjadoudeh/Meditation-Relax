# Meditation-Relax
Meditation-Relax app is an app to help you to have a peaceful moment with your self.

## Architecture

MVVM

## Features

 * Main screen. 
 * Play button take you to player screen.
 * Close Button.
 * Timestamp Slider.
 * Starting and ending dynmic timers.
 * Repeat Button.
 * Backward Button.
 * Play/Pause Button.
  * Forward Button.
 * Stop Button.
